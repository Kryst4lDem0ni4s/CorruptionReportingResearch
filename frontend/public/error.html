Common Issues & Solutions

Cause: The submission ID you entered doesn't exist in the system.

Solution:

  • Double-check the submission ID for typos
  • Ensure you're using the correct ID from your submission confirmation
  • The submission may have been archived (older than 90 days)

Cause: One or more uploaded files exceed the maximum allowed size.

Solution:

  • Images: Maximum 10 MB per file
  • Audio: Maximum 20 MB per file
  • Video: Maximum 50 MB per file
  • Documents: Maximum 5 MB per file
  • Compress files or reduce quality before uploading

Cause: You've exceeded the rate limit of 10 submissions per hour.

Solution:

  • Wait 1 hour before submitting again
  • This limit prevents system abuse
  • If you have multiple pieces of evidence, include them in a single submission

Cause: The submitted data doesn't meet validation requirements.

Solution:

  • Narrative must be at least 50 characters
  • At least one evidence file is required
  • File types must be supported (images, audio, video, PDF)
  • All required fields must be filled

Cause: An unexpected error occurred on the server.

Solution:

  • Wait a few minutes and try again
  • The system may be processing heavy workloads
  • Check if ML models are properly loaded
  • Contact system administrator if error persists

Cause: The backend service is temporarily unavailable.

Solution:

  • The server may be restarting or under maintenance
  • Wait 5-10 minutes and try again
  • Check if the backend Python server is running
  • Verify network connectivity
